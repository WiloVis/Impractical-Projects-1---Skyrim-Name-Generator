# Skyrim Name Generator / Impractical Project-1
A comprehensive, lore-friendly name generator for Elder Scrolls V: Skyrim characters. Perfect for those moments when you've spent 30 minutes on character creation and can't think of a name that fits your Nord warrior or Khajiit thief.

Features
- 10 Playable Races – Nords, Imperials, Bretons, Redguards, Dark Elves (Dunmer), High Elves (Altmer), Wood Elves (Bosmer), Orcs (Orsimer), Khajiit, and Argonians
- 3 Gender Options – Female, Male, and Unisex names for each race 
- Lore-Accurate Names – Hundreds of carefully curated first and last names that fit the Elder Scrolls universe
- Endless Combinations – Randomly generates unique first + last name pairs with each run
- Interactive CLI – Simple menu-driven interface, easy to reroll until you find "the one"
